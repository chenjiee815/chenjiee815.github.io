A Little Java, A Few Patterns： Modern Toys

这一章节，作者主要想通过一系列的对话，让你了解到类型是什么东东。以及如何使用Java定义类型。

类型是什么？

类型是一类值的集合的命名

abstract 、 class 、 extends 各代表什么？

abstract 引入了数据类型

class 引入了变体类型

extends 将以上两者联系起来

Tip

第一条建议

当你想要明确一类数据时，使用 abstract class XXX {} 来定义数据类型

使用 class XXX extends XXX {} 来定义其变体类型
