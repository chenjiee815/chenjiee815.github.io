Flask扩展机制（未完待续）

Date 2014-08-11 21:25:42 By Chen Jie Category python. Tags python flask

Contents

  • Flask扩展导入方式

Flask的扩展包都列在了 Flask扩展注册 中。 这些包本质都属于Python库， 所以你都可以通过 easy_install 或者 pip 来进行安装。

Flask扩展导入方式

Flask扩展的导入方式在Flask 0.8版本前后还是有点区别的。

如果你使用的是Flask 0.8或者其更新的版本，你可以通过：

from flast.ext import ext_name

的方式来导入扩展包。

如果你使用的是比Flask 0.8旧的版本， 你只能通过直接导入扩展包的名称来导入了。

但是Flask官方还是提供了一个兼容模块用来兼容老版本的Flask。 你可以从这里下载： flaskext_compat.py

使用方法:

import flaskext_compat
flaskext_compat.activate()

from flast.ext import ext_name
